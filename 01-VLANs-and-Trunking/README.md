## Objective

Configure VLANs and trunk links to enable communication between multiple switches while maintaining network segmentation.

## Technologies

- Cisco Packet Tracer
- Cisco IOS

## Tasks Performed

- Created VLANs
- Assigned switch ports to VLANs
- Configured trunk ports
- Verified connectivity between devices
- Troubleshot VLAN communication issues

## Skills Demonstrated

- Trunking
- Switch Management
