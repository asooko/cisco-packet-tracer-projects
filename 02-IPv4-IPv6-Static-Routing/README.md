# IPv4 and IPv6 Static Routing

## Overview

This project demonstrates the configuration of IPv4 and IPv6 static and default routes using Cisco Packet Tracer.

## Objectives

- Configure IPv4 addressing
- Configure IPv6 addressing
- Implement static routes
- Configure default routes
- Verify connectivity between networks
- Troubleshoot routing issues

## Skills Demonstrated

- IPv4 Routing
- IPv6 Routing
- Static Route Configuration
- Default Route Configuration
- Router Administration
- Network Troubleshooting

## Tools

- Cisco Packet Tracer
- Cisco IOS
